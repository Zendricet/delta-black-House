三角洲黑屋推测系统是一个基于Web的游戏数据分析工具，旨在帮助玩家评估自己在游戏中的表现和匹配情况。系统通过分析玩家输入的各类游戏数据（如KD值、撤离率等），推测玩家可能遇到的对局类型（如正常局、高压局或黑屋局），并提供隐藏分计算功能。项目完全基于前端技术实现，无需后端支持。

核心功能：

玩家数据输入与分析

五维能力评估（战斗、财富、搜索、生存、合作）

隐藏分计算

黑屋机制原理说明

账号分析报告生成

3. 功能说明
核心功能模块
玩家数据输入

普通KD（绿色）、机密KD（黄色）、绝密KD（红色）

总体撤离率和近十局撤离率

输入框根据数值自动变色提示

五维能力评估（可选）

战斗、财富、搜索、生存、合作能力评分

折叠式设计，节省空间

数据分析

点击"分析数据"按钮进行综合评分

显示进度条动画（约3.5秒）

根据评分显示对局类型：

<400分：唐人/正常对局（蓝色）

400-500分：正常对局（蓝色）

500-700分：高压对局（可能存在外挂玩家）（红色）

700分：黑屋号（黑色）

隐藏分计算

基于总分和五维能力评分计算

未填写五维能力时显示无法计算

账号分析报告

生成详细的账号分析报告

包含所有输入数据和结果

附带对局类型详细说明和黑屋机制分析

其他功能

页面刷新自动清空输入

每5次刷新显示捐赠提示

响应式设计，适配移动设备

6. 使用说明
输入玩家数据

在各输入框中填写对应的游戏数据

KD值会根据数值自动变色提示（红/黄/蓝）

近十局撤离率超过40%时会变红提示

填写五维能力（可选）

展开"五维能力评估"部分

输入0-100的评分值

分析数据

点击"分析数据"按钮

等待3.5秒左右分析完成

查看分析结果和对局类型

查看隐藏分

分析完成后，在"预估隐藏分"区域查看结果

需要填写五维能力才能计算

生成账号报告

分析完成后，点击"生成账号报告"按钮

新窗口将打开详细的分析报告

报告包含所有输入数据和专业分析说明

了解黑屋机制

展开"黑屋机制推测"部分

阅读游戏黑屋机制的详细解释

7. 注意事项
数据准确性

本系统为娱乐性质，分析结果仅供参考

实际游戏机制以官方说明为准

数据安全

浏览器兼容性

推荐使用Chrome、Firefox或Edge等现代浏览器

图片资源

捐赠提示

10. 常见问题
Q：为什么我的分析结果不显示？
A：请确保已填写必要的玩家数据并点击"分析数据"按钮。如果页面卡住，请尝试刷新页面。

Q：为什么隐藏分无法计算？
A：隐藏分需要填写五维能力数据才能计算，请展开"五维能力评估"部分并填写数值。

Q：如何关闭捐赠提示？
A：点击"关闭窗口"按钮即可关闭捐赠提示，下次符合条件（刷新5次）时会再次显示。

Q：报告页面无法正常显示？
A：部分浏览器可能会阻止弹出窗口，请检查浏览器设置允许弹出窗口，或暂时禁用广告拦截插件。

此项目结合了游戏数据分析和前端技术，为用户提供了一个直观、有趣的游戏数据评估工具。所有功能均在浏览器端实现，确保用户数据隐私和安全。
